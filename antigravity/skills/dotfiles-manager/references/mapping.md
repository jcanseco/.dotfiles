# Dotfiles Mapping

> [!IMPORTANT]
> This file is a reference for Antigravity. The **source of truth** for all
> symlink mappings is
> [`~/.dotfiles/scripts/create_links.sh`](../../scripts/create_links.sh).
> Always refer to that script when updating or verifying mappings.

This file defines the mapping between the symlinks in the home directory and the
actual files in the dotfiles repository:

- **Home Path:** `~/.bash_profile`
  **Repo Path:** `~/.dotfiles/shell/bash/bash_profile`
- **Home Path:** `~/.bashrc`
  **Repo Path:** `~/.dotfiles/shell/bash/bashrc`
- **Home Path:**
  `~/.config/sublime-text-3/Packages/User/Preferences.sublime-settings`
  **Repo Path:** `~/.dotfiles/sublime/Preferences.sublime-settings`
- **Home Path:**
  `~/.config/sublime-text/Packages/User/Preferences.sublime-settings`
  **Repo Path:** `~/.dotfiles/sublime/Preferences.sublime-settings`
- **Home Path:** `~/.gemini/GEMINI.md`
  **Repo Path:** `~/.dotfiles/antigravity/GEMINI.md`
- **Home Path:** `~/.gemini/antigravity-cli/settings.json`
  **Repo Path:** `~/.dotfiles/antigravity/settings.json`
- **Home Path:** `~/.gemini/antigravity-cli/skills`
  **Repo Path:** `~/.dotfiles/antigravity/skills`
- **Home Path:** `~/.gitconfig`
  **Repo Path:** `~/.dotfiles/git/gitconfig`
- **Home Path:** `~/.ssh/config`
  **Repo Path:** `~/.dotfiles/ssh/config`
- **Home Path:** `~/.tmux.conf`
  **Repo Path:** `~/.dotfiles/tmux/tmux.conf`
- **Home Path:** `~/.vimrc`
  **Repo Path:** `~/.dotfiles/vim/vimrc`
- **Home Path:** `~/.zshrc`
  **Repo Path:** `~/.dotfiles/shell/zsh/zshrc`
